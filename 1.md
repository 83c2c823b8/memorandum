## wc
ファイルの行数
```sh
  wc -l hoge.txt
```


- ` -w `  : 単語数
- ` -c `  : バイト数
- ` -m `  : 文字数
- ` -L ` : 最長行のながさ

ファイル数
```sh
ls -l | wc -l
```
## デスクトップの参照するディレクトリの変更
HOME/.desktop/ を参照してほしかったので
```
sudo nvim ~/.config/user-dirs.dirs
```
で開いて，
```
XDG_DESKTOP_DIR="$Home/.desktop"
```
と書き換える．
