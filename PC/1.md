# pcまわりの適当な備忘録
## LuaLaTeX 自作パッケージ配置  
```sh
  /usr/share/texlive/texmf-dist/tex/lualatex  
  sudo latexmkrc
```

## カレントディレクトリのパス→クリップボード  
```sh
  pwd | xsel --clipboard --input  
```
clipをエイリアスにしてるので，pwd | clip で可

## wc
ファイルの行数
```sh
  wc -l hoge.txt
```


- ` -w `  : 単語数
- ` -c `  : バイト数
- ` -m `  : 文字数
- ` -L  ` : 最長行のながさ

ファイル数
```sh
  ls -l | wc -l
```
